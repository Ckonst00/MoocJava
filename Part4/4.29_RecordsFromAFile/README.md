## Excercise description

write a program that first prompts the user for the name of the file they want to read. The program then prints the content of the file in the following way

---
Sample data:

lily,3

anton,5

levi,4

amy,1

---
Sample output:

Name of the file:

**data.txt**


lily, age: 3 years

anton, age: 5 years

levi, age: 4 years

amy, age: 1 year
